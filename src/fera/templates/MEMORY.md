# Memory

Persistent facts and preferences.

## Tips

- Use `memory_search(query, mode="quick")` (default) for most lookups.
- Use `memory_search(query, mode="deep")` when quick returns nothing useful, or for ambiguous/nuanced queries about people, past decisions, or recurring topics. Deep mode expands your query into 2-3 alternatives and reranks the top candidates for better recall.
